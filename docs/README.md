## 核心结论速查（多源验证，高置信度）

1. **统计数据 + 来源标注** → AI 可见度提升 30-40%
2. **earned media（第三方内容）** 远优于品牌自有内容
3. **传统 SEO 排名**仍是 LLM 可见性最强预测因子（r=0.65）
4. **不同平台引用逻辑根本不同**，需按平台定制策略
5. **品牌知名度无法被内容优化替代**（Discovery Gap 30:1 鸿沟）
6. **Markdown vs HTML 无显著差异**（Profound A/B 测试 381 页面，2026-01）
7. **早期品牌**：16 次外部提及 > 大量 on-page 优化（OtterlyAI 案例，2026-03）

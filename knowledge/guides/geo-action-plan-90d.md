# GEO 30-60-90 天行动计划

> 从零到有系统的 GEO 能力。按依赖顺序排列，不含时间估算。
> 适用于已有 SEO 基础、想叠加 GEO 能力的内容团队。

---

## Phase 1：地基（前 30 天）

**目标**：确保 AI 能看到你，修复最低成本、最高回报的基础问题。

### 1.1 技术审计（一次性）

- [ ] **robots.txt 白名单**：确认 GPTBot、CCBot、Claude-Web、Google-Extended 未被屏蔽
- [ ] **SSR 确认**：禁用 JavaScript 访问 Top 10 页面，核心内容是否可见？不可见 = 需要 SSR 改造
- [ ] **Bing Webmaster Tools 注册**：ChatGPT Search 基于 Bing 索引，不被 Bing 收录 = 不存在
- [ ] **Schema Markup 部署**：为 Top 10 页面添加 Article/FAQPage/HowTo Schema

**验证**：在 ChatGPT、Perplexity、Google（观察 AIO）中搜索品牌名。能出现 = 技术基础通过。

### 1.2 品牌一致性审计（一次性）

- [ ] 确定 2-3 个品牌核心定位词
- [ ] 审计官网、LinkedIn 公司主页、Wikipedia（如有）、G2/Capterra profiles 中的品牌描述
- [ ] 统一所有渠道的品牌描述——一个字都不能偏
- [ ] 输出：《品牌定位词手册》（一页纸，所有内容创作者人手一份）

### 1.3 存量内容改造（Top 10）

按 `geo-seo-rewrite-guide.md` 的步骤，改造当前流量 Top 10 的页面：

- [ ] 每篇改写引言段为定义性直答
- [ ] 每篇补充至少 3 个统计数据 + 来源标注
- [ ] 每篇添加 FAQ 区域（3-5 个问题）
- [ ] 每篇添加具名作者信息

**Phase 1 完成标志**：技术审计全部通过 + Top 10 页面完成 GEO 改造 + 品牌定位词统一

---

## Phase 2：渠道铺设（30-60 天）

**目标**：在 AI 最看重的第三方渠道建立品牌存在感。

### 2.1 Wikipedia

- [ ] 检查品牌是否有 Wikipedia 词条
- [ ] 有 → 审查准确性，确保与品牌定位一致
- [ ] 没有 → 评估是否满足 Wikipedia 关注度标准（需要独立可靠来源的报道）
- [ ] ⚠️ 不直接编辑自己的词条，通过提供可靠来源让社区编辑完善

### 2.2 Reddit/Quora

- [ ] 识别品牌相关的 Top 5 subreddits
- [ ] 每个 subreddit 以真实用户身份参与讨论（不是发广告）
- [ ] 在 Quora 回答 10 个行业核心问题，引用品牌数据
- [ ] 鼓励真实用户在讨论中分享使用体验

### 2.3 LinkedIn

- [ ] 企业主页：每周发布 1 篇原创知识分享（500-2000 字）
- [ ] 个人 IP：2-3 个核心人物每周发布 2 条行业观点（50-299 字）
- [ ] 内容类型：原创为主（被 AI 引用的 LinkedIn 内容中 95% 为原创）

### 2.4 PR/媒体种子

- [ ] 列出与 OpenAI 有合作协议的媒体清单（路透社、AP、金融时报、Condé Nast 等）
- [ ] 为 1-2 家目标媒体准备思想领导力投稿或专家评论
- [ ] 建立与行业记者的联系

**Phase 2 完成标志**：Wikipedia 词条准确 + Reddit/Quora 有稳定活跃度 + LinkedIn 建立发布节奏

---

## Phase 3：系统化运营（60-90 天）

**目标**：从项目型投入转为可持续的 GEO 运营体系。

### 3.1 内容生产 SOP

- [ ] 所有新增内容使用 `geo-writing-template.md` 模板
- [ ] 发布前必过 `geo-optimization-checklist.md`，≥ 60 分方可发布
- [ ] 建立内容日历：新内容 + 存量更新交替，每季度更新已有内容

### 3.2 监测体系

- [ ] 月度 AI 可见性检查：在 ChatGPT/Perplexity/Google AIO 中搜索 Top 20 目标关键词
- [ ] 记录品牌是否出现、以什么描述出现、引用了哪个页面
- [ ] 对比竞品的 AI 可见性表现
- [ ] ⚠️ AI 引用有波动性（服从幂律分布），至少 3 次检查取中位数

### 3.3 竞品情报

- [ ] 月度检查：在 Perplexity 搜索"最好的 [品类] 工具/产品"，记录 AI 推荐了谁
- [ ] 分析被推荐品牌的内容策略：它们在 Reddit/Wikipedia/LinkedIn 做了什么
- [ ] 找到竞品没做但你能做的差异化切入点

### 3.4 季度原创研究

- [ ] 规划一份行业数据报告或原创研究（年度投入，按季度推进）
- [ ] 研究发布后，在所有渠道交叉推广
- [ ] 原创研究是最高杠杆的 GEO 资产——数据密度内容的 AI 可见度高出 30-40%

**Phase 3 完成标志**：SOP 落地 + 监测体系运转 + 第一份原创研究启动

## 不同规模团队的投入建议

### 1 人团队（创始人/独立顾问）

- Phase 1 全做，但只改造 Top 3 页面
- Phase 2 聚焦 Reddit + LinkedIn 个人 IP（放弃 Wikipedia 和 PR）
- Phase 3 简化：月度自检 + 按模板写新内容即可
- 核心策略：个人品牌 > 企业品牌，LinkedIn 个人创作者在 ChatGPT 中偏好度更高

### 5 人团队

- Phase 1-3 按计划执行
- 分工：1 人负责技术 + 监测，2 人负责内容生产，1 人负责 Reddit/Quora/LinkedIn，1 人负责 PR
- 原创研究用团队内部数据，不需要外部调研预算

### 20 人团队

- Phase 1-3 并行推进（不需要串行）
- 增加：多平台多语言 GEO（中英文独立策略）
- 增加：YouTube 视频内容（占 AIO 引用 18.8%）
- 增加：年度 2-3 份原创研究报告
- 增加：专职 AI 可见性分析师，建立自动化监测 dashboard

## 预期效果

**不承诺具体数字**，但基于研究数据的合理预期：

- Phase 1 后：品牌在 AI 搜索中从"不可见"变为"可被找到"
- Phase 2 后：在部分目标查询中开始出现在 AI 回答中
- Phase 3 后：建立可持续的 GEO 竞争优势，形成内容护城河

**管理期望**：
- GEO 效果不是线性的。品牌知名度是最大变量，内容优化无法弥补知名度空白。
- AI 引用有波动性，不要因为一次搜索没出现就否定整个策略。
- 传统 SEO 排名仍是 LLM 可见性最强预测因子——GEO 不是替代 SEO，是叠加。
